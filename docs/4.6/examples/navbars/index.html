Примеры навигационной панели

Этот пример представляет собой быстрое упражнение, показывающее, как работает панель навигации и ее содержимое. Некоторые панели навигации увеличивают ширину области просмотра, другие ограничиваются рамкой .container. Для позиционирования навигационных панелей проверьте примеры top и fixed top.

В самой маленькой контрольной точке плагин свертывания используется для скрытия ссылок и отображения кнопки меню для переключения свернутого содержимого.
